## 📖 About

This is my personal portfolio website, designed and built from scratch to showcase
my journey as a **Java Full Stack Developer**. It's themed around a code-editor /
IDE aesthetic — file-tree sidebar navigation, syntax-highlighted hero section, and
terminal-style section labels — built entirely with **HTML, CSS, and JavaScript**
(no frameworks, no build tools).

🔗 **Live:** [jayeshkumardhariyal.github.io/personal-portfolio-website](https://jayeshkumardhariyal.github.io/personal-portfolio-website/)

## 🚀 Features

- 🎨 **Editor-inspired UI** — file-tree sidebar nav, terminal labels, syntax-highlighted hero code block
- 📱 **Fully responsive** — adapts cleanly from mobile (hamburger nav) to desktop
- 🧭 **Scroll-spy navigation** — sidebar auto-highlights the active section as you scroll
- 🎬 **Scroll-reveal animations** — sections fade/slide in on scroll using `IntersectionObserver`
- 🖱️ **Custom cursor** — subtle accent-colored cursor dot on desktop (disabled on touch devices)
- ♿ **Accessible** — semantic HTML, focus states, and respects `prefers-reduced-motion`
- 🌗 **No dependencies** — loads instantly, no frameworks or build tools required

## 🖥️ Running Locally

```bash
# Clone the repo
git clone https://github.com/jayeshkumardhariyal/personal-portfolio-website.git

# Move into the folder
cd personal-portfolio-website

# Open directly in a browser
open index.html        # macOS
start index.html        # Windows

# OR serve locally (recommended)
python3 -m http.server 8000
# then visit http://localhost:8000
```
